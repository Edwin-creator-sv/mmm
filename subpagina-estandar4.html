Estándar 4: IEEE-802.11

Es una familia de especificaciones técnicas desarrolladas por el IEEE para implementar redes de área local inalámbricas (WLAN). Es el estándar que define el Wi-Fi.

  • Característica Principal: Define las capas físicas y de control de acceso al medio para la comunicación inalámbrica, permitiendo que dispositivos electrónicos se conecten a una red sin cables físicos.
  • Aplicación: Es la base tecnológica para todas las redes Wi-Fi, permitiendo la conectividad inalámbrica en hogares, oficinas, espacios públicos y entre dispositivos.
  • Beneficios: Proporciona flexibilidad y movilidad al usuario, elimina la necesidad de cableado físico, permite la conexión de múltiples dispositivos a una misma red y facilita la interoperabilidad entre diferentes fabricantes.
