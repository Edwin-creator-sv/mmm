Estándar 3: W3C

El W3C es la principal organización internacional de estandarización para la World Wide Web. Es un consorcio internacional donde organizaciones miembro, personal a tiempo completo y el público trabajan juntos para desarrollar estándares web.

  • Característica Principal: Crea estándares abiertos y no propietarios para la Web, como HTML, CSS, XML, y las WCAG. Su objetivo es asegurar la interoperabilidad de la Web.
  • Aplicación: Desarrollo y mantenimiento de tecnologías web fundamentales para garantizar que la web sea accesible para todos los usuarios y dispositivos, y que sea robusta y funcional.
  • Beneficios: Promueve una Web abierta y accesible, fomenta la innovación, garantiza la compatibilidad entre navegadores y dispositivos, y proporciona las bases tecnológicas para el funcionamiento de Internet.
